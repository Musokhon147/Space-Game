# Space Game (GME201 Final)

This is a Unity project for a Space Game, developed as part of the GME201 Final assignment.

## Project Structure

The project is structured as a standard Unity project:
- **Assets**: Contains all game assets, scripts, scenes, and resources.
  - **Scripts**: Game logic and behaviors.
  - **Scenes**: Game levels (gameplay, menus, cutscenes).
  - **Directors**: Timeline and cinematic assets.
  - **Settings**: Project settings and configuration.

## Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Musokhon147/Space-Game.git
    ```
2.  **Open in Unity:**
    - Open Unity Hub.
    - Click "Add" and select the cloned project folder.
    - Open the project (ensure you have a compatible Unity version installed).
